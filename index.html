Intelligence Engine v1.2

Mapping the Atlassian Neural Network

v1.2.0 Stable

NeuraJ is a specialized administrative auditing suite designed to bridge the visibility gap in Jira Cloud administration. By synthesizing data from GraphQL Ecosystem nodes and Public Automation REST APIs, it provides a comprehensive map of instance health and technical debt.

⚡ Automation Intelligence

Scans the public Automation engine to track rule status, owner account IDs, and execution health. Essential for identifying high-consumption rules.

🔌 Add-on Audit Pro

Connects via GraphQL to categorize apps into System, Commercial, or Marketplace-Paid tiers, enabling precise license cost auditing.

🧩 Technical Debt Mapping

Identifies "Field Bloat" by mapping custom fields to screen schemes, flagging any fields not actively assigned to a visual context.

🔄 Workflow Optimization

Calculates step-to-transition ratios to identify complex "Spaghetti Workflows" that hinder user productivity and instance performance.

👥 Active User Count

Filters through managed directories to provide an accurate count of billable seats versus inactive or system accounts.

🔒 Zero-Persistence Security

Privacy-first design. All retrieved data remains in browser memory. No Jira metadata is ever stored on external NeuraJ servers.

Administrative Compliance

Usage of NeuraJ implies the operator has authorized access to metadata. This tool is "Read-Only" and does not modify Jira configurations; it only synthesizes existing data for optimization purposes.
